**Tweet 3 (7 Layers)**

The 7 layers:

1. Working Memory (active focus)
2. Short-Term (session)
3. Episodic (experiences)
4. Long-Term Semantic (facts + FSRS)
5. Procedural (skills)
6. Core Memory (pinned facts)
7. Cross-Context (shared knowledge)

Each with different retention, consolidation, and retrieval rules.
